# UI Capability Alignment

The UI now prioritises the Cognee hackathon story:

- MemoryMesh is a broad work-memory layer for long-running agents.
- The coding agent is the proof demo, not the only use case.
- Cognee lifecycle operations are visible: remember, recall, improve, forget.
- The recovery brief is the main winning moment.

The console can still evolve into a broader enterprise view, but the submission-facing flow should lead with the coding-agent proof because it gives objective evidence of recovery.
