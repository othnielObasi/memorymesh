# Demo Script

## 30-second pitch

AI agents do not only forget conversations; they forget the work. MemoryMesh gives long-running agents durable work memory using Cognee, so they can recover after context loss, continue from the right checkpoint, and improve from past outcomes.

## 2-minute demo flow

1. Open the MemoryMesh coding-agent proof UI.
2. Explain that coding is only the proof case; the product works for any long-running workflow.
3. Select **Open-source Cognee**.
4. Click **Run selected backend proof**.
5. Show tests failing before the patch.
6. Show Cognee `remember`: task contract, repo map, decision, failure signal.
7. Show simulated context loss.
8. Show Cognee `recall`: recovery brief restored.
9. Show the patch diff.
10. Show tests passing after recovery.
11. Show Cognee `improve`: future recovery lesson saved.
12. Click **Forget this demo memory** to show Cognee `forget` cleanup.
13. Select **Cognee Cloud** and repeat, or click **Run OSS + Cloud proof**.

## Closing line

MemoryMesh does not give agents longer chat history. It gives them durable memory of work, and it can run on both open-source Cognee and Cognee Cloud.
