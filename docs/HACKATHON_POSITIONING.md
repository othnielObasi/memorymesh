# Hackathon Positioning

## Name

MemoryMesh

## Tagline

Never-forget work memory for long-running AI agents.

## Category

Primary: Never-Forget Workflows  
Secondary: Self-Improving Agents

## Core differentiator

Most submissions will likely show assistants that remember users. MemoryMesh shows agents that remember work: the task, decisions, evidence, failures, checkpoint, and next action.

## Demo proof

The included coding-agent proof performs real local work:

- inspect repo
- run tests
- save Cognee memory
- simulate context loss
- recall recovery brief
- patch code
- rerun tests
- improve future recovery
- forget session memory on demand

## Expansion beyond coding

The same memory model applies to research agents, support agents, operations workflows, compliance reviews, and enterprise automations.
